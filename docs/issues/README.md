# Engineering Issues

本目录按 GitHub 风格记录当前工程问题和待治理事项：

- [P0](p0/README.md)：阻断发布、造成数据丢失或存在紧急安全风险的问题；
- [P1](p1/README.md)：高优先级正确性、可靠性或生产运维风险，应安排在近期处理；
- [P2](p2/README.md)：一般缺陷、体验问题或计划中的能力补齐，不阻断当前核心流程；
- [P3](p3/README.md)：低优先级优化、体验改进或非关键维护事项；
- [P4](p4/README.md)：长期规划、实验性想法或当前暂不考虑的事项。

当前没有待修复的 P0 问题。每个 issue 只保留当前状态、影响范围、复现/背景和验收条件；已解决问题不继续占用活动 issue 列表。

## 分类
