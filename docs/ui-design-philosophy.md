# UI Design Philosophy

OpenExcel 的 UI 设计遵循 **克制留白 + 圆润亲和 + 图标引导** 的核心理念。

## 核心原则

### 1. 大面积留白
- 页面底色纯白，内容不堆满
- 留白即层级，用空间而非边框分割信息
- 视觉重心聚焦于核心内容，减少视觉噪音

### 2. 克制圆角
- 常规容器和控件使用 4–16px 的圆角，保持层级清晰
- `pill shape`（`border-radius: 999px`）只用于适合的按钮、标签和状态控件
- 避免为了统一而让所有元素都变成胶囊形

### 3. 图标与文字清晰表达
- 主要功能操作使用 icon + label 配对呈现
- 图标按钮必须有明确的语义或 tooltip
- 图标颜色用于表达层级和状态，不要求每个图标都使用彩色

### 4. 卡片承载内容
- 功能模块用白色卡片（轻微阴影或边框）组织
- 卡片与背景有明确但克制的视觉区分
- 卡片标题字号与正文形成清晰梯度

### 5. 低饱和点缀色
- 界面以黑白灰为主色调
- 彩色仅用于图标、重点文字、交互态
- 避免大面积色块，防止视觉疲劳

### 6. 三级字体层次
- **大标题**（16–24px）：页面标题、对话问候语
- **卡片标题**（12–14px）：模块名称、输入框标签
- **正文**（11–13px）：内容文本、状态提示
- 梯度清晰，用户一眼可知信息权重

### 7. 辅助信息弱化
- 版权、版本、法律提示和状态信息使用次要视觉层级
- 辅助信息不应抢占主要工作区，也不应遮挡内容

### 8. 自适应与稳定尺寸
- 布局使用 flex、grid 和响应式约束，避免依赖脆弱的固定位置
- 在不同尺寸屏幕上保持合理的内容比例和可用空间
- 工具栏、按钮、输入框等关键交互区域应保持稳定尺寸，不因文本或视口变化而跳动

## 视觉系统参数

| 参数 | 值 |
|------|-----|
| 背景色 | `#FFFFFF` |
| 边框色 | `#e8eaed` |
| 卡片边框 | `1px solid #e8eaed` |
| 文本主色 | `#1f1f1f` |
| 文本次色 | `#666`, `#999` |
| 点缀色（蓝） | `#5b9bd5` |
| 圆角（小） | 8px |
| 圆角（中） | 12px |
| 圆角（大/pill） | 999px |
| 输入框高度 | 自适应，min 100px |
| 头像尺寸 | 26px |
| 字体 | 系统默认无衬线 |
