# UI Design Philosophy

OpenExcel 的 UI 设计遵循 **克制留白 + 圆润亲和 + 图标引导** 的核心理念。

## 核心原则

### 1. 大面积留白
- 页面底色纯白，内容不堆满
- 留白即层级，用空间而非边框分割信息
- 视觉重心聚焦于核心内容，减少视觉噪音

### 2. 大圆角药丸化
- 所有交互元素统一大圆角（8–16px），无尖锐边缘
- 按钮、标签、切换开关一律 pill shape（`border-radius: 999px`）
- 视觉亲和、现代、无压迫感

### 3. 图标 + 文字显式表达
- 每个功能操作均 icon + label 配对呈现
- 不依赖纯图标按钮，避免用户猜测含义
- 图标本身为低饱和彩色（蓝/紫/橙/青），起引导作用

### 4. 卡片承载内容
- 功能模块用白色卡片（轻微阴影或边框）组织
- 卡片与背景有明确但克制的视觉区分
- 卡片标题字号与正文形成清晰梯度

### 5. 低饱和点缀色
- 界面以黑白灰为主色调
- 彩色仅用于图标、重点文字、交互态
- 避免大面积色块，防止视觉疲劳

### 6. 三级字体层次
- **大标题**（16–24px）：页面标题、对话问候语
- **卡片标题**（12–14px）：模块名称、输入框标签
- **正文**（11–13px）：内容文本、状态提示
- 梯度清晰，用户一眼可知信息权重

### 7. 固定底部信息行
- 版权、版本、法律提示等固定在底部
- 灰字小字（10–11px），不抢主内容焦点

### 8. 自适应与比例
- 布局元素使用相对单位（`vw`、`flex`）而非固定像素
- 在不同尺寸屏幕上保持合理的左右比例
- 关键交互区域（如输入框）大小不随视口缩放而变形

## 视觉系统参数

| 参数 | 值 |
|------|-----|
| 背景色 | `#FFFFFF` |
| 边框色 | `#e0e4ea` |
| 卡片边框 | `1px solid #e8eaed` |
| 文本主色 | `#1f1f1f` |
| 文本次色 | `#666`, `#999` |
| 点缀色（蓝） | `#5b9bd5` |
| 圆角（小） | 8px |
| 圆角（中） | 12px |
| 圆角（大/pill） | 999px |
| 输入框高度 | 自适应，min 100px |
| 头像尺寸 | 26px |
| 字体 | 系统默认无衬线 |
